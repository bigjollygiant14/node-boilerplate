# Node Boilerplate 0.0.1

Node Server Boilerplate


## Prerequisite
1. Node
2. NPM
3. Gulp
4. Bower


## Installation
1. Clone
2. `npm install`
3. `bower install`


## Run Locally
1. `gulp` (compile SASS, run server then watch for changes) or `gulp start` (run server)


## Dependencies
1. angular latest
2. bootstrap sass latest
3. lodash latest


## Test
`gulp test`
